# DARC - Defense Analyses and Research Corporation

A dual-personality website featuring a deliberately boring DC think-tank facade that transforms into an edgy cyberpunk terminal aesthetic.

## Quick Start

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) - you'll see the boring homepage.

## The Duality

### Homepage (/)
- Ultra-boring DC think-tank aesthetic
- Navy blue, gray, white colors
- Generic mission statement
- Fake board members
- Contains hidden Easter eggs

### All Other Pages
- Full DARC mode: cyberpunk/NGE terminal aesthetic
- Black background, red/cyan/green accents
- CRT scanlines and glitch effects
- Japanese warning labels (機密, 警告)
- All 30 real DARC articles

## Easter Eggs

Try to find these hidden gems:

1. **Type "darc"** anywhere on the boring homepage to permanently unlock DARC mode
2. **Konami Code**: ↑↑↓↓←→←→BA
3. **Hover** over "Corporation" in the footer
4. **Check** the copyright year
5. **Click** the period after "Inc." 5 times
6. **Inspect element** and delete `#normie-curtain`
7. Look for the publication titled "Strategic Privatization in Modern Deterrence Theory"
8. Check the hero image alt text
9. Open the favicon in an image editor and increase brightness +70%
10. View page source for ASCII art
